# Toolbox for Unity

A collection of utilities and reusable code I use in my prototypes and other projects. Amassed over time via tutorials and guides.

# Installation

To install the latest version of this package copy this url and add it using [the package manager UI](https://docs.unity3d.com/Packages/com.unity.package-manager-ui@2.1/manual/index.html):

`https://github.com/mactinite/UnityToolbox.git#main`

# Development

Add as an embedded package to a unity project using git submodules.

`git submodule add https://github.com/mactinite/UnityToolbox.git Packagese/Toolbox`
